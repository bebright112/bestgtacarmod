Installation

Single Player Installation

1. Open OpenIV → Enable Edit Mode

2. Drag the DLC folder into:

update/x64/dlcpacks/

3. Edit dlclist.xml and add:

<Item>dlcpacks:/23vlmdh/</Item>

FiveM Installation

1. Place the vehicle folder inside:

/resources/[cars]/

2. Add this to server.cfg:

ensure 23vlmdh

Spawn Code

23vlmdh
